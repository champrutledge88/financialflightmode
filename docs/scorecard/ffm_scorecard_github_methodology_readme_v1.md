# Flight Plan Budget Scorecard Methodology

## Purpose

This folder documents the Financial Flight Mode scoring methodology for the Flight Plan Budget Scorecard.

The Scorecard is designed to give pilots a fast financial control check using simple monthly inputs.

The public experience must stay simple:

1. Enter numbers.
2. Receive score.
3. Understand status.
4. Take three next moves.

## Current Methodology Principle

The detailed scoring logic lives behind the cockpit.

The pilot should not be shown long financial explanations or complex formulas.

## Core Score Signals

The Financial Flight Score evaluates:

- Cash Flow Control
- Savings System
- Emergency Runway
- Debt Load
- Wealth Fuel

## Important Labeling Note

Do not label a metric as Debt-to-Income unless the code calculates formal DTI.

Formal DTI is generally monthly debt payments divided by gross monthly income.

If the code compares total debt balance to monthly income, use the label Debt Load.

## Score Categories

Recommended current categories:

| Score | Status |
|---:|---|
| 0 to 49 | Pre-Flight |
| 50 to 74 | Stabilizing |
| 75 to 89 | Cruise Control |
| 90 to 100 | Flight Mode |

If the live website still uses Grounded, Takeoff Ready, Cruising, and Flight Mode, document that as the current front-end label set before changing.

## Public Copy Standard

Every score explanation should be:

- Short
- Clear
- Practical
- No shame
- No hype
- No more than three recommended actions

## Evidence Sources

Use trusted financial education sources to validate thresholds:

- Federal Reserve household financial well-being reports
- CFPB emergency savings guidance
- CFPB DTI education
- CFPB Financial Well-Being Scale
- BLS Consumer Expenditure Survey
- Fidelity savings benchmark
- Fannie Mae DTI guidance context

## Change Control

Before changing the live formula:

1. Capture current scoring code.
2. Document current formulas.
3. Validate labels.
4. Run household test scenarios.
5. Update methodology notes.
6. Review public copy.
7. Commit with clear version notes.
8. Deploy only after review.

## Education-Only Position

The Financial Flight Score is an educational diagnostic. It is not a credit score, lending decision, investment recommendation, tax advice, legal advice, or personal financial plan.
