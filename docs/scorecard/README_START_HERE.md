# FFM Scorecard MD Files

Start here.

This folder contains the Markdown documentation for the Flight Plan Budget Scorecard validation system.

Recommended reading order:

1. `ffm_scorecard_methodology_sop_v1.md`
2. `ffm_scorecard_public_explanation_copy_v1.md`
3. `ffm_scorecard_github_methodology_readme_v1.md`
4. `ffm_scorecard_validation_backlog_v1.md`

Core rule:

The internal methodology can be detailed, but the pilot-facing score explanation must stay simple, clear, and quick.

Do not overload the public Scorecard with long financial explanations.
